بحر: آزاد

جانے کو ہے وہ رشکِ قمر دیکھ رہا ہوں

میں شام سے آثارِسحر دیکھ رہا ہوں

ہر روز انھیں غیر کے گھر دیکھ رہا ہوں

یہ جذبِ محبت کا اثر دیکھ رہا ہوں

جلوے ہیں یہہ آنکھوں میں میری کون و مکاں کے

یا وسعتِ داماںِنظر دیکھ رہا ہوں

سمجھاتھا جسے دوست میں نکلا وہی دشمن

کچھ رنگ زمانے کا دگر دیکھ رہا ہوں

کامل ہو اگر عشق تو بندھتا ہے تصور

آگے وہ نہیں میرے مگر دیکھ رہا ہوں

بستی ہی میں لگتا ہے نہ صحرامیں مرا دل

دنیا سے الگ رہنے کو گھر دیکھ رہا ہوں

سچ کہتے ہیں دم بھر میں گذرتی ہے شبِ وصل

میں شام سے آثارِ سحر دیکھ رہا ہوں

حیران ہوں کس طرح سمایا ترا جلوہ

میں وسعتِ دامانِ نظر دیکھ رہا ہوں

جنبش ہوئی اس ابروئے پیوستہ کو یکتاؔ

کھینچی ہوئی پھر تیغ دوسر دیکھ رہا ہوں
